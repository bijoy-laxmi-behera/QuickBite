# 🍔 QuickBite - MERN Stack Food Ordering App

## 🚀 Tech Stack
- React (Vite)
- Node.js
- Express
- MongoDB
- JWT Authentication
- OTP Password Reset

## 📁 Project Structure
- quickbite-frontend → React App
- quickbite-backend → API Server

## ⚙️ Setup

### Backend
cd quickbite-backend
npm install
npm run dev

### Frontend
cd quickbite-frontend
npm install
npm run dev
