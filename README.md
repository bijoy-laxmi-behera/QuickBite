# 🍔 QuickBite - MERN Stack Food Ordering App

QuickBite is a full-stack MERN food ordering application where users can browse food items, add products to cart, place orders, and make secure online payments.

---

# 🚀 Live Demo

### Frontend

https://quick-bite-2026.vercel.app/

### Backend API

https://quickbite-backend-dpuf.onrender.com/

---

# 🛠️ Tech Stack

## Frontend

* React.js
* Vite
* React Router DOM
* Axios
* Tailwind CSS

## Backend

* Node.js
* Express.js
* MongoDB Atlas
* JWT Authentication
* OTP Password Reset
* Razorpay Payment Gateway
* Cloudinary Image Upload

---

# ✨ Features

* 🔐 JWT Authentication
* 📧 OTP Password Reset via Email
* 🛒 Add to Cart Functionality
* 💳 Razorpay Payment Integration
* 📦 Order Management
* 🖼️ Cloudinary Image Upload
* 📱 Responsive UI
* ⚡ Fast API Communication

# ⚙️ Environment Variables

## Backend (.env)

```env
PORT=4000

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_jwt_secret

EMAIL_USER=your_email@gmail.com
EMAIL_PASS=your_app_password

CLOUD_NAME=your_cloudinary_name
CLOUD_API_KEY=your_cloudinary_key
CLOUD_API_SECRET=your_cloudinary_secret

RAZORPAY_KEY=your_razorpay_key
RAZORPAY_SECRET=your_razorpay_secret
```

---

## Frontend (.env)

```env
VITE_API_URL=https://quickbite-backend-dpuf.onrender.com/api

VITE_RAZORPAY_KEY=your_razorpay_key
```

# 📦 Installation & Setup

## Clone Repository

```bash
git clone https://github.com/bijoy-laxmi-behera/QuickBite.git
```

---

# Backend Setup

```bash
cd backend
npm install
npm run dev
```

---

# Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
